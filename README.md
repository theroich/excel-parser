# excel-parser
Parse a excel file to a Java Bean.

## Install 
Use maven to install in your local repository:
```
mvn clean install
```

Then add this dependency to pom:
```
<dependency>
   <groupId>net.thrch</groupId>
   <artifactId>excelParser</artifactId>
   <version>0.0.2</version>
</dependency>
```
